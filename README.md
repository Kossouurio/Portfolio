# Portfolio Website

This portfolio website showcases projects and work done by Kossouurio. The website is built using plain HTML, CSS, and JavaScript, with a focus on a black and yellow color scheme.

## Project Structure

The project is organized as follows:

```
Portfolio
├── css
│   ├── style.css        # Main styles for the website
│   └── reset.css        # CSS reset for consistent styling across browsers
├── js
│   ├── main.js          # Main JavaScript functionality
│   └── github.js        # GitHub API interactions to fetch projects
├── assets
│   └── fonts            # Directory for font files
├── index.html           # Main entry point of the website
├── projects.html        # Displays projects from GitHub
├── about.html           # Information about Kossouurio
├── contact.html         # Contact form or information
└── README.md            # Project documentation
```

## Setup Instructions

1. Navigate to this link:


## Features

- Responsive design that adapts to different screen sizes.
- Dynamic loading of projects from GitHub using the GitHub API.
- Clean and modern layout with a focus on usability.
